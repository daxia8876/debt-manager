# 债务管家 Android App

## 📱 如何编译 APK（零基础版）

### 第一步：注册 GitHub 账号
1. 打开 https://github.com
2. 点击右上角 **Sign up** 注册免费账号
3. 验证邮箱完成注册

---

### 第二步：创建仓库并上传文件

1. 登录后点击右上角 **"+"** → **New repository**
2. Repository name 填：`debt-manager`
3. 选择 **Public**（公开，免费编译）
4. 点击 **Create repository**

然后上传文件：
1. 点击页面中的 **uploading an existing file**
2. 将本压缩包解压后，把**所有文件和文件夹**拖进去
3. 点击 **Commit changes**

---

### 第三步：等待自动编译（约 5 分钟）

1. 点击仓库页面的 **Actions** 标签
2. 可以看到 **Build APK** 正在运行（黄色转圈）
3. 等变成 ✅ 绿色勾

---

### 第四步：下载 APK

1. 点击那个绿色的 ✅ **Build APK**
2. 页面下方 **Artifacts** 部分
3. 点击 **债务管家-V14** 下载 ZIP
4. 解压得到 `.apk` 文件

---

### 第五步：安装到手机

1. 将 APK 文件传到手机（微信/QQ/数据线均可）
2. 手机打开文件管理器找到 APK
3. 点击安装（需要允许"安装未知来源应用"）
4. 完成！

---

## ⚠️ 常见问题

**Q: Actions 没有运行？**
A: 检查文件是否上传到了 `main` 或 `master` 分支

**Q: 安装时提示"未知来源"？**
A: 进入手机设置 → 安全 → 允许安装未知来源 → 再安装

**Q: 数据会丢吗？**
A: 数据存在手机本地，卸载 App 才会清除
